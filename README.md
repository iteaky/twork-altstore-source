# Twork Beta

Private beta source for AltStore Classic testers.

## Tester Instructions

1. Install AltStore Classic using AltServer.
2. Open AltStore on iPhone.
3. Go to Sources.
4. Add this source URL:
   `https://iteaky.github.io/twork-altstore-source/source.json`
5. Install Twork.
6. Refresh the app every 7 days through AltStore.

## Public Files

- `source.json`
- `icon.png`
- `screenshots/`
- `scripts/update_source.sh`

## Publishing Notes

- Keep the private Twork source repository closed.
- Do not add app source code or private build artifacts here.
- Regenerate `source.json` whenever a new beta IPA is released.
